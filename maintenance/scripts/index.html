IOF MRO Glossary

Posted on September 18, 2023

Property Natural Language Definition
 has maintenance state inverse of 'stateOf'
 initiates comes before an event or process in time and results in beginning or creation of the event or process
 maintenance state of relation that describes the maintenance state of a maintainable item
Class Natural Language Definition
 degraded state state of reduced ability to perform as required but with acceptable reduced performance
 disposition to exhibit undesirable behavior disposition of an item to exhibit undesirable behavior
 disposition to fail disposition of an item to undergo a failure process or exhibit undesirable behavior
 failed state state of an item being unable to perform a required function due to a failure event
 failure effect process that is the consequence of failure, within or beyond the boundary of the failed item
 failure event event that causes an item to lose its ability to perform a required function
 failure mode code information content entity that describes an undesireable disposition
 failure process process that changes some quality of an item causing the the item to become degraded or failed
 functioning process process that enables an item to perform a function
 maintenance activity maintenance process that is a single task to retain or restore the function of a maintainable item
 maintenance process process to do with retaining or restoring the function of a maintainable item under a maintenance strategy
 maintenance state stasis that holds during a temporal interval when the realizable functions and capabilities of the participating item, or the grade of realization of those functions and capabilities, remain unchanged
 maintenance strategy maintenance (method/ approach/ actions) to enable (the/an) asset to achieve (management's/ desired) objectives
 maintenance work order record information content entity that describes a maintenance process
 operating state state of an item being able to perform a required function assuming that the external resources, if required, are provided
 person qualification specification requirement specification that identifies the need for a person to have an assessed skill for a specific task
 qualified maintenance person person who is qualified to perform a specified specified maintenance process or task
 required function function of a maintainable item which is considered necessary to fulfil a process requirement
 supporting maintenance activity single action in support of the execution of a maintenance process
 undesireable disposition disposition of an item to undergo a process that leads to undesirable outcomes
